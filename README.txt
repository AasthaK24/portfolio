# Aastha Portfolio

A single-file portfolio site built with Tailwind (CDN).

## How to run locally
Open `index.html` in your browser.

## Deploy (GitHub Pages)
1. Create a new repo (e.g., `aastha-portfolio`).
2. Upload `index.html` and `resume.pdf`.
3. In repo Settings → Pages → Deploy from `main` branch `/root`.
4. Open the Pages URL.

## Deploy (Netlify)
1. Go to https://app.netlify.com → Add new site → Deploy manually.
2. Drag-and-drop the folder with `index.html` and `resume.pdf`.
3. Done — Netlify gives you a public URL.
